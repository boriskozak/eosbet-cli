# eosbet-cli
> play eosbet in terminal

## Installation
### From npm
```bash
npm i -g @yelo/eosbet-cli
```

### From git
```bash
git clone git@github.com:imyelo/eosbet-cli.git
cd eosbet-cli
npm i
npm link
```

## Usage
### Config
```bash
eosbet config
```

### Play dice
```bash
eosbet dice
```
